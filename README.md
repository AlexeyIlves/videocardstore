# GPU Store

Modern frontend pet project of an online GPU store built with vanilla JavaScript, HTML and CSS.

The project simulates a real e-commerce experience with:

* product catalog
* product pages
* shopping cart
* checkout flow
* filters and search
* responsive layout
* localStorage persistence

## Features

### Product Catalog

* Responsive product grid
* Product cards with stock status
* GPU specifications
* Search by title/specs/brand
* Filtering by brand and memory size
* Price sorting

### Product Page

* Detailed GPU information
* Specifications table
* Stock availability
* Add to cart functionality

### Shopping Cart

* Quantity controls
* Dynamic total price calculation
* Remove products
* localStorage persistence

### Checkout

* Delivery form
* Payment method selection
* Order summary
* Form validation
* Order completion flow

### UI/UX

* Responsive design
* Toast notifications
* Smooth animations
* Modern card-based layout
* Mobile-friendly interface

---

## Tech Stack

* HTML5

* Vanilla JavaScript

## Installation

Clone repository:

```bash
git clone https://github.com/YOUR_USERNAME/gpu-store.git
```

Open project folder:

```bash
cd gpu-store
```

Run locally:

```bash
open index.html
```

or use Live Server in Visual Studio Code.
